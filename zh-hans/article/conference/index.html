HMOs细胞工厂构建面临的挑战 #

  • 糖基转移酶底物特异性低；
  • 前体合成途径协同调控难；
  • 关键调控靶点不清晰。

设计-构建-测试-学习(DBTL)酶分子改造迭代框架的开发 #

  • 开发了一种集成了分子动力学模拟、量子力学计算以及结构生物学分析等多学科研究策略的设计-构建-测试-学习(DBTL)迭代框架，能够提高酶的底物特异性与催化活性。

多靶点双重碱基编辑的MultiduBE的构建 #

  • 基于CRISPR/dCas12a构建可实现多靶点双重碱基编辑的MultiduBE(Multiplex dual Base Editor)，实现对于特定基因的靶向原位(in-situ)诱变，可用于构建酶的突变体文库。

多尺度综合代谢网络模型etiBsu1209的构建 #

  • 基于搭建的自动化模型搭建软件Model-Tool，构建了集成酶约束、热力学约束和转录调控网络的多尺度综合代谢网络模型etiBsu1209[1]，实现了多目标产物的精准预测，准确率74.3%。 [1] etiBsu1209: A comprehensive multiscale metabolic model for Bacillus subtilis. DOI

基于多组学机器学习的综合代谢网络模型的构建 #

  • 基于搭建的高质量多组学数据集，构建了34种机器学习的组合模型，进一步构建了多尺度综合代谢网络模型(iBsu1209-ME)[2]，实现了基因表达与细胞生长的精准预测，准确率87.9%。 [2] A multi-omics, machine learning-aware, genome-wide metabolic model of Bacillus subtilis refines the gene expression and cell growth prediction. DOI

温敏型单输入多输出基因线路和双向基因线路的构建 #

  • 基于转录因子CI857，构建了温敏型的表达调控元件，进一步构建了温敏型的单输入多输出基因线路和双向基因线路，能够响应温度时序性的激活和抑制靶基因的表达。

可编程多功能生物分子凝聚体的从头设计 #

  • 基于液液相分离机制，通过“搭积木”策略从头设计系列人工无序蛋白(SIDP)，并进一步利用其构建胞内人工生物分子凝聚体，实现了代谢网络与蛋白翻译过程的正交调控。

基于级联基因线路的组合调控与超高通量筛选 #

  • 利用级联基因线路实现了“代谢物-荧光”信号的转换放大与代谢途径的精细组合调控，基于crRNA阵列文库构建了5184种调控优化组合，通过“流式初筛-自动化复筛”获得高产菌株。

基于级联基因线路的组合调控与超高通量筛选 #

  • 利用级联基因线路实现了“代谢物-荧光”信号的转换放大与代谢途径的精细组合调控，基于crRNA阵列文库构建了5184种调控优化组合，通过“流式初筛-自动化复筛”获得高产菌株

基于全细胞传感器的突变体高通量筛选 #

  • 通过构建2’-FL全细胞生物传感器，结合液滴微流控系统，成功筛选获得能够使2’-FL产量提高1.8倍，且无副产物DFL生成的FutC突变体。

2’-岩藻糖基乳糖(2’-FL)细胞工厂的构建与优化 #

  • 通过异源GDP-岩藻糖合成途径的组装适配、辅因子与前体的供应强化等策略，实现了以葡萄糖/蔗糖和乳糖为底物的2’-FL从头合成。
  • 开发了一个能够拓宽转录因子底物谱的筛选平台，获得了一系列能够特异性响糖的转录因子突变体。
  • 基于乳糖响应型转录因子，构建了一个响应乳糖浓度动态平衡细胞生长与前体乳糖合成的基因线路，2’-FL在3-L罐上的产量达30.1 g/L(葡萄糖唯一碳源)
  • 以大肠杆菌MG1655为出发菌株，通过合成途径优化重构、多拷贝异源a-1,3-岩藻糖基转移酶FutM,，强化关键酶的可溶性表达，实现了3-FL的从头合成
  • 构建基于无序蛋白的人工无膜细胞器系统，并将其应用于3-FL合成途径酶的共定位组装，有效提升了3-FL的产量，摇瓶水平菌株3-FL产量为8.2 g/L
